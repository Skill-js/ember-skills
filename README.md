## Overview

This repository provides a massive, production-quality collection of Agent Skills specifically designed for AI coding assistants working with **Ember.js**. 

Ember has evolved significantly with the Octane and Polaris editions. It is crucial for AI agents to understand the differences between modern Ember patterns (Glimmer components, template tags, `@tracked`, WarpDrive) and legacy patterns (`Ember.Object`, Mixins, classic components). This collection bridges that gap.

## Skills Included

- **`ember-components`**: Modern Glimmer components and `.gjs`/`.gts` Template Tags.
- **`ember-routing`**: Route architecture and navigation.
- **`ember-services`**: Dependency injection and state management.
- **`ember-resources`**: Managing async and reactive state cleanly.
- **`ember-data`**: Modern WarpDrive, RequestManager, and Cache paradigms.
- **`ember-testing`**: QUnit, `@ember/test-helpers`, and isolated component testing.
- **`ember-tooling`**: Generators, Embroider, and Glint setup.
- **`ember-concurrency`**: Managing complex asynchronous tasks and race conditions.
- **`ember-mirage`**: Modern API mocking setup.
- **`ember-forms`**: Handling form state and headless forms natively.
- **`ember-legacy`**: Crucial guardrails helping agents avoid legacy patterns.

## How to Use

If your AI coding agent supports the [AgentSkills.io specification](https://agentskills.io), simply point it to the specific skill directory you need, or load the entire collection into your agent's workspace.
